1. Wymienić oraz opisz dwie części selektora,

    2. Część przed dwukropkiem (:), zwana selektorem elementu (ang. element selector), która określa, jakiego rodzaju elementy HTML chcemy stylizować, na przykład p dla akapitów, h1 dla nagłówków pierwszego poziomu itp.
    Część po dwukropku (:), zwana deklaracją stylów (ang. declaration block), która zawiera właściwości CSS, takie jak kolor tekstu, tło, marginesy itp. oraz wartości tych właściwości.

  2. Znaczenie nawisów klamrowym {...},

    3. W języku CSS nawiasy klamrowe {} są wykorzystywane do definiowania bloków deklaracji stylów, czyli do określenia, jakie właściwości CSS mają zostać zastosowane dla określonych selektorów.

  3. Opisz cechy,

    4. Selektory to elementy języka CSS, które umożliwiają określenie, jakie elementy HTML mają zostać stylizowane.

  4. Opisz wartości

    5. Selektor elementu - wybiera wszystkie elementy HTML danego typu. Przykładowe wartości to p dla akapitów, h1 dla nagłówków pierwszego poziomu, a dla linków itp.
    Selektor klasy - wybiera elementy, które mają określoną klasę. Przykładowe wartości to .header, .container, .button itp.
    Selektor identyfikatora - wybiera element, który ma określony identyfikator. Wartość to #header, #main, #footer itp.
    Selektor atrybutu - wybiera elementy, które mają określony atrybut lub wartość atrybutu. Przykładowe wartości to [href] dla elementów z atrybutem href, [type="text"] dla elementów z atrybutem type równym text.
    Selektor grupowy - umożliwia łączenie kilku selektorów w jedno wyrażenie. Wartości to, na przykład, p, h1, h2 dla stylizowania akapitów, nagłówków pierwszego i drugiego poziomu.
    Selektor potomny - wybiera elementy, które są potomkami określonego elementu HTML. Wartość to, na przykład, div p dla stylizowania wszystkich akapitów, które są potomkami elementów div.
    Selektor pseudoklasy - wybiera elementy na podstawie ich stanu lub zachowania. Przykładowe wartości to :hover, :active, :focus itp.

  5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny

    6. W języku CSS znak średnika (;) służy do oddzielania poszczególnych deklaracji stylów. Znak ten powinien być stosowany po każdej deklaracji stylu, aby oddzielić ją od kolejnej.

    W niektórych sytuacjach znak średnika nie jest konieczny, na przykład:
    w przypadku ostatniej deklaracji stylu w bloku CSS. W tym przypadku znak średnika nie jest wymagany, ale jego dodanie nie wpłynie negatywnie na kod.

    w przypadku jednej deklaracji stylu, np. dla nadania koloru tła dla całej strony. W tym przypadku znak średnika nie jest wymagany, ale wartość ta nie może być łączona z innymi deklaracjami stylu.
